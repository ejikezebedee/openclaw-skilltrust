# Security Policy

OpenClaw SkillTrust is a defensive project for reviewing AI-agent extension
trust boundaries.

## Supported Scope

Please report issues that affect:

- Trust scoring correctness
- Unsafe false negatives
- Secret masking failures
- Report integrity
- Quarantine bundle integrity
- Manifest parsing safety

## Out of Scope

This project does not accept requests or contributions for exploitation,
credential harvesting, stealth, persistence, evasion, unauthorized access, or
offensive automation.

## Reporting

Open a private security advisory if available, or create a minimal public issue
without secrets, credentials, or exploit instructions.
